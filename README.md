# Summoning Scroll

A Minecraft mod that add Scroll to summon an entity.

## Description

This mod add an item called Summoning Scroll to summon an entity.
A scroll need a catalyst to be in an offhand to determine which entity
to summon, which can be change in the config file.

## Required Dependencies

| Mod                | Version   | Link                               |
| ------------------ | --------- | ---------------------------------- |
| Minecraft NeoForge | 21.1.226+ | [Download](https://neoforged.net/) |
